# 申请授权

本章节描述了如何授权为应用程序访问组织创建或购买的资产数据。


## 授权申请<authorization>

1. 单击EnOS控制台左侧导航面板中的 **IAM > 申请授权**。

2. 单击要授予权限的应用程序后的 |edit| 。

3. 在 **资产授权** 页面中。

   - 左侧的资产树包括根节点和所有子节点。勾选子节点前的复选框，将选择从属于该节点的所有站点。
   - 右侧显示当前所选节点下所有站点的信息以及站点位置的路径。

.. note:: 选中“所有资产”选项后，默认情况下，将为当前应用程序授权访问的任何新站点。
